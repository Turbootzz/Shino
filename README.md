# Shino
 Javascript Discord bot.


## Installation
 Initialise project
 ```
 npm init -y
 ```

 Install DiscordJS
 ```
 npm install discord.js
 ```

 Install DOT Enviroment
 ```
 npm install dotenv
 ```

  Install rest API:
 ```
 npm install discord.js @discordjs/rest
 ```

 Install Nodemon for dev refreshes.
 ```
 npm i --save-dev nodemon
 ```

## Start the bot
 For production:
 ```
 npm start
 ```
 
 For developer mode:
 ```
 npm run start:dev
 ```
