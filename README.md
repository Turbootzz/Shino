# Shino
 Javascript Discord bot.


## Installation
 Initialise project
 ``` npm init -y ```

 Install DiscordJS
 ``` npm install discord.js ```

 Install DOT Enviroment
 ``` npm install dotenv ```

 Install Nodemon for dev refreshes.
 ``` npm i --save-dev nodemon ```

 To start the bot:
 ``` npm start ```
 
 Dev start:
 ``` nodemon bot.js ```

 Install rest API:
 ``` npm install discord.js @discordjs/rest ```
